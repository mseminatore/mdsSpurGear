# mdsSpurGear

Simple OpenSCAD script for parametric spur gear generation.  

Based on Boston Gear design rules https://www.bostongear.com/pdf/gear_theory.pdf

The script has several options including:

1. Number of teeth
2. Diametral pitch
3. Pressure angle
4. Shaft diameter
5. Gear thickness
6. Major holes
7. Minor holes
8. Number of holes
